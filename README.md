# Surf 4 Fun — мини Discord (голосовая комната)

Голосовое приложение на WebRTC с поддержкой до **5 участников** в одной комнате и **отдельной регулировкой громкости** каждого участника для каждого пользователя.

## Стек

- **Frontend:** Vue 3, Vite, TypeScript, Tailwind CSS, Shadcn-vue (radix-vue + CVA)
- **Backend:** Node.js, Socket.IO (сигналинг для WebRTC)
- **Голос:** WebRTC (mesh), Web Audio API (GainNode для громкости)

## Запуск

1. Установка зависимостей:
   ```bash
   npm install
   ```

2. Запуск сервера сигналинга (в одном терминале):
   ```bash
   npm run server
   ```

3. Запуск фронтенда (в другом терминале):
   ```bash
   npm run dev
   ```

4. Откройте http://localhost:5173, введите имя и ID комнаты (например `room-1`), нажмите «Войти». Откройте ту же комнату в другой вкладке/браузере — участники подключатся; у каждого будет свой слайдер громкости для каждого другого участника.

## Запуск через HTTPS (локально)

Для теста по HTTPS (например, доступ к микрофону не с localhost): браузер общается только с Vite по HTTPS, а запросы к сокетам проксируются на бэкенд по HTTP — **на порту 8000 HTTPS не нужен**.

1. **Сервер** (один терминал) — обычный HTTP:
   ```bash
   npm run server
   ```
   Работает на **http://localhost:8000**.

2. **Фронтенд с HTTPS** (второй терминал):
   ```bash
   npm run dev:https
   ```
   Сайт: **https://localhost:5173**. Запросы к `/socket.io` Vite проксирует на `http://localhost:8000`.

3. При первом заходе браузер может предупредить о сертификате — нажмите «Дополнительно» → «Перейти на сайт».

**Продакшен:** HTTPS настраивается на платформе (Vercel, Netlify, Railway и т.д.) или через reverse proxy (nginx/Caddy) с Let's Encrypt.

## Функции

- До 5 человек в одной комнате
- Регулировка громкости каждого участника отдельно (0–100%) для каждого слушателя
- Вкл/выкл микрофона
- Выход из комнаты

## Структура

- `server/index.js` — Socket.IO сервер (порты, офферы/ответы/ICE)
- `src/composables/useVoiceRoom.ts` — логика комнаты: WebRTC, громкость через GainNode
- `src/components/VoiceRoom.vue` — UI комнаты и слайдеры громкости
- `src/components/JoinForm.vue` — форма входа (имя, ID комнаты)
- `src/components/ui/*` — Shadcn-vue: Button, Slider, Card, Input
